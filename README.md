# open_excuse_letter_bad_weather
This repository is intended to provide sattelite imagery visuals sourced from [JMA](https://www.data.jma.go.jp/mscweb/data/himawari/sat_img.php?area=se2).  No malicious content is associated with this landing page. Its sole purpose is to inform Filipinos about ongoing scientific projects and promote preparedness for national calamities.

# FYI
**No scraping. No unauthorized API tricks. Every Himawari-8 Band 13 image here is captured programmatically and served via serverless Node.js on Vercel CDN — built clean, compliant with PH RA 8293, 10173, and 10175, and made with respect for our Japanese counterparts.**
